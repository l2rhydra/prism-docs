## Client Exports

### Show Notification
Show a notification with type,msg,duration

```txt copy
prims_Notification_show
```

`[required array ex:{msg="hello", type="success", time=5000}]`

### Hide specific notification
Hide a specific notification

```txt copy
prims_Notification_delete
```

### Hide all notifications
Hide all notifications

```txt copy
prims_Notification_delete_all
```

### Keybind

```lua copy
exports.prims_hud:prims_keybind_show({
  key = "E",
  title = "Inventory",
  description = "Press E for open inventory"
})

--for hide: 
exports.prims_hud:prims_keybind_hide()
```
