## Configuration
Add this to your `server.cfg` file:

```shell
setr paintball_avatar_api_key  "API KEY HERE"
```

For people to use the avatar system, for example on Discord or Steam, they must write this command in the server.cfg file. A Steam API key is required for Steam, and a Discord bot token is required for Discord.

## Installation

- **Ensure all dependencies are installed** [ [ox_lib](https://github.com/communityox/ox_lib/releases) ]
- **Edit configuration as desired** (Optional)
- **Drag-and-drop our script** : Drag `prism_paintball`, `prism_paintball_map`, `prism_paintball_gun`, `prism_paintball_flag`  in to your `[prism_paintball]` directory inside your `resources` directory and add `ensure [prism_paintball]` to your `server.cfg` after all dependencies.
- **Restart your server**.
